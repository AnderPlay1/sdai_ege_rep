
    <h2>Задание 23 Теория</h2>
    <div class="theory">
    <p>Вот как решаются задачи на подсчёт количества маршрутов в графе:</p>
    <pre>   <strong>1.Прочти задачу</strong>
        Выясни из условия задачи:

        -какие команды выполняет исполнитель
        -каково исходное число и какое число должно быть результатом
        -через какие числа должна проходить траектория, а через какие — не должна</pre>
    <pre>   <strong>2.Создай электронную таблицу для расчётов</strong>
        В заголовках строк перечисли команды исполнителя. Добавь строку «Всего», в которой будет рассчитываться количество программ, которые переводят исходное число в каждое число из диапазона.
        В заголовках столбцов напиши все числа из диапазона от исходного числа до итогового.</pre>
    <pre>   <strong>3.Заполни строку «Всего»</strong>
        Для исходного числа введи количество программ 1. Для всех остальных чисел из диапазона введи формулу, которая находит сумму количеств команд в строках.</pre>
    <pre>   <strong>4.Заполни строки для каждой команды</strong>
        Если число Х в заголовке столбца можно получить из какого-то числа Y с помощью команды Z, введи в ячейку на пересечении столбца Х и строки команды Z ссылку на ячейку в строке «Всего»,
        которая находится в столбце Y.</pre>
    <pre>   <strong>5.Обработай условие отсутствия некоторого числа Х во всех траекториях</strong>
        Если траектория вычислений не должна проходить через число X, удали число в строке «Всего» в столбце Х</pre>
    <pre>   <strong>6.Обработай условие присутствия некоторого числа Х во всех траекториях</strong>
        Представь, что столбец Х — это первый столбец таблицы. Удали из столбцов, следующих за столбцом Х, числа, которые нельзя получить из Х с помощью каждой из команд.</pre>
    <pre>   <strong>7.Ответь на вопрос задачи</strong>
        Ответ находится в последней ячейке строки «Всего».</pre>
</div>
