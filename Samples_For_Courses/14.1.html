Задание 14 Теория

Вот как решаются задачи на вычисление арифметического выражения в других системах счисления:

   1.Прочти задачу
        Посмотри на основания систем счисления, в которых находятся числа в выражении.
        Отметь, какое значение неизвестной цифры тебе нужно найти, наименьшее или наибольшее.
        Определи, что нужно указать в ответе. Чаще всего это частное от деления значения выражения, в которое подставлена неизвестная цифра, на указанное в задаче число.
        Важно! Если основания систем счисления меньше либо равны 36, проще всего использовать функцию int(), чтобы перевести числа в десятичную систему счисления.
   2.Составь строку с алфавитом системы счисления
        Количество символов должно быть равно основанию системы счисления.
        Сначала в строке идут цифры от 0 до 9, затем — латинские буквы в алфавитном порядке.
        Например, для девятнадцатеричной системы счисления алфавит будет выглядеть так:

            '0123456789ABCDEFGHI'

Важно! Порядок цифр и букв должен строго соблюдаться. Его можно проверить с помощью названий столбцов в любой электронной таблице.

   3.Создай цикл для перебора всех возможных значений неизвестной цифры
        В этом цикле неизвестная цифра должна принимать значения всех символов в строке с алфавитом системы счисления.
        Например, если x — это неизвестная цифра, а система счисления девятнадцатеричная, нужно написать так:

            for x in '0123456789ABCDEFGHI':

Следующие действия делай в цикле.

   4.Преобразуй каждое число из выражения в строку
        С помощью операции склеивания строк + соедини известные части числа, взятые в одинарные кавычки, с переменными, которые обозначают неизвестные цифры.
   5.Преобразуй полученные строки в десятичные числа
        Воспользуйся функцией с двумя параметрами: int(строка с числом для перевода, основание системы счисления).
   6.Проверь, выполняется ли условие задачи, с помощью оператора if
        Если условие задачи выполняется, выведи результат.
   7.Ответь на вопрос задачи
        Если выводится несколько ответов, нужно выбрать тот, который соответствует условию задачи.
        Можешь вывести неизвестную цифру, для которой этот ответ получен, дополнительно к ответу.
        Затем выбери ответ, найденный для минимального или максимального значения неизвестной цифры, в зависимости от условия.
