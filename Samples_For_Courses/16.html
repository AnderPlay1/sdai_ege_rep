Задание 16 Теория

Вариант решения 1

Чтобы решить задание, надо расписывать значения функции F(n) с конца, от тех значений n, которые нужны для вычисления выражения. Это помогает быстрее упростить выражение и избежать лишних вычислений.

  1.Изучи условия задачи и выбери начальную точку
    Определи, какие значения функции нужно найти, чтобы вычислить выражение.
    Например, если требуется вычислить F(54) − F(50), имеет смысл начать с F(54) и последовательно двигаться назад,
    чтобы выразить значения F(53), F(52) и так далее, пока не станет понятно, как они взаимодействуют.
  2.Последовательно вычисли значения функции от нужного n назад
    Начни с самого большого n, которое нужно для выражения, и последовательно двигайся к меньшим значениям.
    Для каждого n используй соответствующее правило, чтобы выразить F(n) через предыдущее значение. В процессе можно увидеть, что часть выражений упрощается или сокращается.
  3.Выяви закономерности и сокращения в выражении
    Постепенно можно заметить, что некоторые слагаемые или множители сокращаются. Это помогает быстрее прийти к окончательному числу или к упрощённому виду выражения.
  4.Подставь конечные значения и вычисли ответ
    Когда все нужные значения функции для выражения найдены, подставь их и упрости результат, чтобы получить окончательное число.

Вариант решения 2

Вот как решаются задания на работу с рекурсиями с помощью языка программирования Python:

  1.Прочти условие задачи
    Проанализируй алгоритм вычисления значения функции:

    -Узнай, каково условие завершения рекурсии. Это базовый случай, при котором результат вычисления функции становится равным какому-то конкретному значению (числу или строке).
    -Определи, как описываются правила получения новых значений из базового случая и уже вычисленных ранее значений.
    -Выясни, какое выражение или алгоритм с использованием рекурсивной функции нужно составить в основной программе.
  2.Напиши заголовок функции:

    def имя функции (параметр функции):
  3.Определи условие завершения рекурсии
    В блоке команд функции определи условие завершения рекурсии. В нём нужно описать базовый случай. Если его не задать, рекурсия будет работать бесконечно.
    Условие завершения рекурсии описывается в формате:

    if (условие завершения рекурсии):
	    return конкретное значение функции  
  4.Определи рекуррентные формулы
    В блоке команд функции определи одну или несколько рекуррентных формул, которые задают правила получения новых значений из базового случая и уже вычисленных ранее значений.
    Формат описания рекуррентной формулы:

    if (условия применения правила):
	    return описание правила получения новых значений
  5.Вызови функцию в основной программе
    Напиши выражение или алгоритм, в котором вызывается рекурсивная функция. Рекурсивная функция многократно вызывает саму себя с параметрами, вычисленными по правилам,
    которые указаны в рекуррентной формуле. Вызовы происходят до тех пор, пока функция не будет вызвана для базового случая и не найдётся конкретное значение.
  6.Увеличь глубину рекурсии при необходимости
    Информация о цепочке вызовов сохраняется в структуре, которая называется «стек вызовов». Размер стека вызовов ограничен в среде программирования.
    Если максимальный размер достигнут, а базовый случай ещё не вычислен, программа выдаст ошибку maximum recursion depth exceeded in comparison.
    Её можно попробовать исправить, увеличив глубину рекурсии c помощью функции setrecursionlimit() из модуля sys.
  7.Используй мемоизацию рекурсии при необходимости
    В некоторых случаях функции приходится вычислять одни и те же промежуточные значения несколько раз. Каждый раз это делается заново, потому, что промежуточные значения не сохраняются.
    От этого процесс вычислений замедляется. Чтобы его ускорить, организуй мемоизацию рекурсии, т. е. сохранение в памяти значений промежуточных вычислений.
    Мемоизацию можно сделать с помощью декоратора @lru_cache из модуля functools или с использованием списка.
  8.Ответь на вопрос задачи
    Посмотри ещё раз условие задачи и проверь, правильно ли составлено выражение или алгоритм с рекурсивной функцией. После проверки, запиши ответ в поле для результатов.
