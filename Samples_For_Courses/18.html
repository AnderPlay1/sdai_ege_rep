Задание 18 Теория

Вот как решаются задачи на использование Робота в электронных таблицах:

   1.Прочти задачу
Выясни из условия задачи:
    • какая клетка — начальная, а какая — конечная точка маршрута; если конечных точек может быть несколько, определи их признаки
    • какие команды может выполнять Робот
    • в какой последовательности записывать в ответе найденные суммы: сначала максимальную, а затем минимальную, или наоборот
   2.Скопируй исходную таблицу, чтобы создать таблицу для расчётов
Вставь копию ниже исходной таблицы, пропустив одну строку. Если в исходной таблице не выделены границы, залей фон копии любым цветом.
Важно! Если не определить границы таблицы для расчётов, можно случайно растянуть формулы на большее количество строк, чем необходимо.
   3.Определи суммы, которые может собрать Робот в начальном столбце и в начальной строке таблицы
Начальные строка и столбец пересекаются в начальной клетке маршрута. Число монет в начальной клетке расчётной таблицы равно числу монет в начальной клетке исходной таблицы. Для остальных клеток начального столбца и начальной строки вычисляй сумму по правилу: сумма равна количеству монет в аналогичной клетке исходной таблицы плюс сумма в ячейке расчётной таблицы, из которой Робот попал в эту клетку.
   4.Определи максимальные суммы в остальных клетках таблицы
Для всех клеток, кроме клеток начального столбца и начальной строки, вычисляй сумму по правилу: сумма равна количеству монет в аналогичной клетке исходной таблицы плюс максимальная из сумм в ячейках расчётной таблицы, из которых Робот может попасть в эту клетку. Для определения максимума используй функцию МАКС().
Важно! При копировании формул в ячейки таблицы будут стёрты стены. Чтобы их восстановить, скопируй и вставь форматирование исходной таблицы.
   5.Измени формулы в клетках, попаданию в которые мешают стены
Выдели цветом те клетки, в которые можно попасть только с одного направления (второе направление перекрыто стеной). Измени формулы в выделенных ячейках так, чтобы не использовать значение из ячейки, которая находится за стеной.
   6.Измени формулы в клетках, попаданию в которые мешают стены
Выдели цветом те клетки, в которые можно попасть только с одного направления (второе направление перекрыто стеной). Измени формулы в выделенных ячейках так, чтобы не использовать значение из ячейки, которая находится за стеной.
   7.Определи итоговые суммы в возможных конечных точках маршрута
В зависимости от условий задачи конечная точка маршрута может быть одна или их может быть несколько. Выдели конечные точки маршрута цветом. Найди конечную точку с максимальной суммой.
   8.Реши задачу для поиска минимальной суммы
Выдели расчётную таблицу и нажми CTRL+H (замена текста в документе). Замени слово “МАКС” на “МИН”. Формулы изменятся и в ячейках появятся минимальные суммы. Найди конечную точку с минимальной суммой.
   9.Ответь на вопрос задачи
Запиши найденные максимальную и минимальную суммы в том порядке, который указан в условии задачи.
